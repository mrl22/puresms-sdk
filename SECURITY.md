# Security policy

## Supported versions

Security fixes are applied to the latest released version on `main`.

## Reporting a vulnerability

Please use GitHub’s private security-advisory reporting flow for this repository. Do not include PureSMS API keys, webhook secrets, phone numbers, or message content in a public issue.

Reports should include a clear reproduction, affected version, impact, and any proposed mitigation. Maintainers will acknowledge receipt and coordinate a fix and disclosure timeline through the private report.
